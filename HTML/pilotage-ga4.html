Pilotage Taleos

Graphiques et tableaux liés à Google Analytics 4 (candidatures, événements extension). Une page unique, sans script local.

Configurer l’intégration (une minute)

  1. Ouvrez Looker Studio et créez un rapport avec la source Google Analytics 4 (propriété Taleos).
  2. Menu Partager → Intégrer le rapport → copiez l’URL d’intégration (commence par https://lookerstudio.google.com/embed/...).
  3. Dans le fichier HTML/pilotage-ga4.html, collez cette URL dans window.TALEOS_LOOKER_EMBED_URL (en haut du fichier), enregistrez et déployez.

L’API Data GA4 ne peut pas être appelée directement depuis une page web publique sans serveur (les identifiants resteraient visibles). Looker Studio affiche les mêmes données GA4 dans une page intégrable, sans terminal ni variable d’environnement.
